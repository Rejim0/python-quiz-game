# Python Quiz Game

This is a simple True/False quiz game I built using Python while learning object-oriented programming.

The program asks a series of questions and the user has to answer **True or False**. After each question, the program tells whether the answer is correct and shows the current score. At the end, it shows the final score.

## What I practiced in this project

- Python classes and objects
- Using multiple Python files
- Importing modules
- Loops and condition checking
- Basic user input

## Project Files

- `main.py` – runs the quiz
- `quiz_brain.py` – controls quiz logic
- `question_model.py` – creates question objects
- `data.py` – contains the quiz questions
